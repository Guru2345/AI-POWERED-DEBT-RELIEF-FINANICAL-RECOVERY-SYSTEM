# 🌟 AI-Powered Debt Relief System - Project Profile Drive

This is a standalone project profile page showcasing the **AI-Powered Debt Relief Financial Recovery System** project.

## 🚀 Quick Start

### Option 1: Open Locally
Simply open `index.html` in your web browser to view the showcase page.

### Option 1: Deploy to GitHub Pages
1. Create a new repository for your project profile
2. Upload the `index.html` file
3. Enable GitHub Pages in repository settings
4. Access your project profile at: `https://yourusername.github.io/your-repo-name`

### Option 2: Deploy to Netlify (Recommended)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the project-profile-drive folder
3. Get instant live URL

### Option 3: Deploy to Vercel
1. Go to [vercel.com](https://vercel.com)
2. Import from Git or upload project-profile-drive folder
3. Deploy instantly
