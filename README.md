## ⚡ **Quick Start Guide**

### **Prerequisites**
- 📦 Node.js 18+ (Latest LTS recommended)
- 🔑 Google Gemini API Key
- 💻 Modern web browser with ES2022+ support

### **Installation**

```bash
# Clone the repository
git clone https://github.com/Guru2345/AI-POWERED-DEBT-RELIEF-FINANICAL-RECOVERY-SYSTEM.git

# Navigate to project directory
cd AI-POWERED-DEBT-RELIEF-FINANICAL-RECOVERY-SYSTEM

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env
# Add your GEMINI_API_KEY to the .env file

# Start development server
npm run dev
```

### **🌐 Access Your Platform**
- **Live Production**: https://ais-dev-kqnufxjnn62rwtactzdxtm-550408133837.asia-southeast1.run.app
- **Development**: http://localhost:3000
- **Production Build**: `npm run build && npm start`
